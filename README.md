# brief
Brief makes your url's as `brief` as the word: `brief`

## Features
- [x] User Authentication
- [x] Hashing Algorithm (MD5 + base64 URL Encoding)
- [ ] API Testing
- [ ] Caching with Redis
- [ ] Analytics
- [ ] Swagger Documentation
- [ ] Postman Documentation

# Requires
`go 1.17+` `postgresql` `redis` `docker (optional)`
